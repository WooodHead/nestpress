# use-favicon

## Installation

```bash
$ yarn (or `npm install`)
```

## Development Mode

```bash
$ yarn dev (or `npm run dev`)
```

Go to `http://localhost:3000`, and you'll see `Next.js on top of NestJS!`.

## Production Mode

```bash
$ yarn build (or `npm run build`)
$ yarn start (or `npm start`)
```
